\# OpenStack Integration



Computus Cloud is powered by \*\*OpenStack\*\*, one of the most widely used open-source cloud platforms.



\## What This Means for You



\- Full OpenStack API compatibility (advanced users)

\- Standard OpenStack concepts (flavors, images, security groups, etc.)

\- Proven, enterprise-grade infrastructure

\- S3-compatible object storage layer on top of Swift



\## Beta Specifics



\- Limited to 2 VMs and 200 GB S3 storage

\- Public IPv6 only (no IPv4 floating IPs yet)

\- Single project/tenant model

\- OSIE-style dashboard on top for simplified user experience



This architecture allows us to rapidly improve and expand the platform based on your beta feedback.
