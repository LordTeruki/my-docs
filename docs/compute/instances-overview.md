\# Instances Overview (Beta)



The Compute section allows you to create and manage virtual machines on Computus Cloud Beta.



\## Beta Compute Limits



\- Maximum \*\*2 virtual machines\*\* total per account

\- Only \*\*Tiny\*\*, \*\*Small\*\*, and \*\*Medium\*\* flavors are available

\- Available images: \*\*Ubuntu\*\*, \*\*Debian\*\*, and \*\*Rocky Linux\*\*

\- Public access: \*\*IPv6 only\*\* (no public IPv4 addresses)

\- Private IPv4 networking is fully supported



\## Key Points



\- You must upload at least one SSH key before launching a VM

\- All VMs are billed hourly while running (metered only — no real charges in beta)

\- VMs receive a public IPv6 address automatically when launched on the external network



\*\*Tip:\*\* We are very interested in your feedback on performance, boot times, and networking behavior.
