\# Available Images (Beta)



The following official images are available:



\- \*\*Ubuntu\*\* — Latest stable version (recommended for most users)

\- \*\*Debian\*\* — Latest stable version

\- \*\*Rocky Linux\*\* — Latest stable version (RHEL compatible)



These are clean, minimal images with cloud-init support for SSH key injection.



\*\*Note:\*\* Custom image upload is not available during the beta.
