\# Reboot, Stop \& Resize



\## Available Actions



| Action     | Description |

|------------|-----------|

| \*\*Reboot\*\*     | Soft or Hard reboot of the VM |

| \*\*Stop / Shut Off\*\* | Power off the VM (still billed while stopped) |

| \*\*Start\*\*      | Power on a stopped VM |

| \*\*Resize\*\*     | Change flavor (Tiny ↔ Small ↔ Medium) |



\*\*Note:\*\* Resize may require the VM to be stopped first.



\## Recommendations



\- Stop your VMs when not in use to reduce compute usage (even though no real charges in beta)

\- Always make sure you have important data backed up before resizing
