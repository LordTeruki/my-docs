# What is Computus Cloud Beta?

**Computus Cloud** is an independent cloud computing platform built on **OpenStack**, designed to provide reliable virtual machines, private networking, and S3-compatible object storage.

This **Beta** version allows early testers to try the platform before public launch. The beta runs on our production-grade OpenStack deployment with **OSIE** handling user accounts, billing, metering, and service workflows.

## Key Features in Beta

- Virtual machines (limited to 2 per account)
- S3-compatible object storage (max 200 GB)
- Full private networking capabilities
- IPv6 public access
- Pay-as-you-go metering through OSIE

## Why We're Running a Beta

We want real users to test the platform, discover edge cases, and give feedback on usability, performance, and features before opening it to everyone.

Your participation helps us improve stability, documentation, and the overall experience.
