# First Login to beta.computus.io

After verifying your email:

1. Go to https://beta.computus.io
2. Log in with your email and password
3. You will land on the main dashboard

## Recommended First Actions After Login

1. **Create S3 Credentials**
   - Go to **Access → S3/EC2 Credentials**
   - Click **Create Credentials**
   - Save your **Access Key** and **Secret Key** securely

2. **Upload an SSH Public Key**
   - Go to **Access → SSH Keys**
   - Click **Create SSH Key** and paste your public key

3. **Review Your Limits**
   - Check current usage under Billing

4. Start testing:
   - Create your first S3 bucket
   - Launch your first VM (max 2 total)

## Dashboard Tips

- Use the left sidebar to navigate between Compute, Networking, Object Storage, etc.
- Monitor your usage regularly in the Billing section
- Use the Support Requests page to send feedback
