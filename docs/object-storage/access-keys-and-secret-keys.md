# Creating Access & Secret Keys

You need an Access Key and Secret Key to use the S3 API.

## How to Create Credentials

1. Log in to https://beta.computus.io
2. In the left menu, go to **Access → S3 Credentials**
3. Click **Create Credentials** / **Create S3 Key**
4. Enter a description (e.g. "AWS CLI Key" or "Cyberduck")
5. Click **Create**
6. **Immediately copy** the following:
   - **Access Key**
   - **Secret Key**

> Save these securely. The Secret Key will not be shown again.

## Important Notes

- You can create multiple key pairs
- Each key works with all your buckets
- Rotate or delete keys regularly for security
