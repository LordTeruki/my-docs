# S3 Endpoint (Beta)

## Endpoint Details

- **URL**: `https://s3.computus.io:8080`
- **Port**: `8080` (important — do not omit it)
- **Protocol**: HTTPS
- **Compatible with**: AWS CLI, s3cmd, Cyberduck, S3 Browser, rclone, boto3, etc.

**Note:** This is different from the production endpoint. Use `:8080` during beta.
