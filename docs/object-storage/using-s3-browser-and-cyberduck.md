\# Using S3 Browser and Cyberduck (Beta)



\## Recommended Tools



\- \*\*S3 Browser\*\* (Windows) — simple and free

\- \*\*Cyberduck\*\* (Windows / macOS) — user friendly

\- \*\*rclone\*\* — great for scripting



\### Cyberduck Setup Example



1\. Open Cyberduck

2\. Click \*\*Open Connection\*\*

3\. Choose \*\*Amazon S3\*\*

4\. Server: `s3.computus.io`

5\. Port: `8080`

6\. Access Key ID: \[your key]

7\. Secret Key: \[your key]

8\. Click \*\*Connect\*\*
