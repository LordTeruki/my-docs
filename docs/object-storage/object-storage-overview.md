\# S3 Object Storage Overview (Beta)



Computus Cloud provides fully \*\*S3-compatible\*\* object storage, perfect for backups, media storage, application data, and more.



\## Beta Limits

\- \*\*200 GB\*\* maximum total storage per account

\- Usage is metered daily

\- No enforcement — you can go over temporarily, but please stay close to the limit



\## Quick Start Summary



1\. Create S3 credentials in the dashboard

2\. Create a bucket

3\. Test with AWS CLI, S3 Browser, Cyberduck, or any S3 client



\*\*Endpoint:\*\* `https://s3.computus.io:8080`
