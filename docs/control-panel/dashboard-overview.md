\# Dashboard Overview



The Computus Cloud dashboard is your central hub for managing all resources in the beta.



\## Main Sections



| Menu Item              | Description |

|------------------------|-----------|

| \*\*Dashboard\*\*          | Overview of your usage, active VMs, storage quota, and quick actions |

| \*\*Compute\*\*            | Manage virtual machines (Instances) |

| \*\*Networking\*\*         | Create and manage networks, subnets, routers, and security groups |

| \*\*Object Storage\*\*     | S3 credentials and storage overview |

| \*\*Access\*\*             | SSH Keys and S3 Credentials |

| \*\*Billing\*\*            | Usage, invoices, payments, and credits |

| \*\*Projects\*\*           | Project and tenant management |

| \*\*Support\*\*            | Submit feedback and support requests |



\## Key Dashboard Widgets (Beta)



\- Current VM count (out of 2 allowed)

\- S3 storage usage (out of 200 GB)

\- Active instances and their IPv6 addresses

\- Recent billing summary



\*\*Tip:\*\* Bookmark https://beta.computus.io for quick access.
