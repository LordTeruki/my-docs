# Projects and Tenants

During the beta, Computus Cloud uses a **single-project model** for simplicity.

## How It Works

- When you sign up, a project is automatically created for you.
- All your resources (VMs, networks, S3 buckets, etc.) belong to this single project.
- You cannot create additional projects during the beta.

## Project Details

You can view your project information by going to **Projects** in the left menu.

Typical information shown:

- Project ID
- Project Name
- Creation date
- Current quotas and usage

**Note:** Multi-project support will be added after the beta.
