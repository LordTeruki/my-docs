# Endpoint Reference (Beta)

## Main Endpoints

Beta Dashboard: https://beta.computus.io  
S3 Object Storage: https://s3.computus.io:8080

## Important Notes

- Always use https://s3.computus.io:8080 for S3 operations during beta
- No public IPv4 addresses — use IPv6 for VM access
- Port 8080 is required for S3 during beta
