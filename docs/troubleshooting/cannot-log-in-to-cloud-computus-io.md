# Cannot Log In to beta.computus.io

## Common Solutions

1. Email Verification
   - Check your spam or junk folder for the verification email
   - Request a new verification link from the login page

2. Password Reset
   - Use the Forgot Password link on the login page

3. Browser Issues
   - Try incognito or private mode
   - Clear cache and cookies
   - Try a different browser (Chrome recommended)

4. Account Not Approved
   - Beta access may require manual approval — contact support if you signed up but cannot log in
