# Networks Overview (Beta)

Computus Cloud Beta gives you full control over networking. You can create private networks and attach them to your VMs as needed.

## Beta Networking Highlights

- Public access: **IPv6 only** (automatic when using "external net")
- Private IPv4 networking: Fully supported and unlimited
- You can create as many private networks and subnets as you want
- We are especially interested in creative private networking setups during beta

## Main Networking Components

- **Private Networks** — Isolated Layer 2 networks
- **Subnets** — IP address ranges inside networks
- **Routers** — For connecting networks and enabling outbound access
- **Security Groups** — Firewall rules for your VM
