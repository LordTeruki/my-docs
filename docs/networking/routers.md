\# Subnets \& Routers



\## Creating a Subnet



1\. Go to the network you created

2\. Click \*\*Create Subnet\*\*

3\. Set:

&#x20;  - Subnet Name

&#x20;  - Network Address (CIDR) — e.g. `192.168.10.0/24`

&#x20;  - Gateway IP (optional)



\## Creating a Router



Routers allow traffic between networks or to the external network.



1\. Routers will not be avaible during this Beta



\*\*Note:\*\* During beta, IPv6 is the primary public connectivity method.
