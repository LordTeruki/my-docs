# Security Groups

Security Groups act as virtual firewalls for your VMs.

## Default Security Group

- Allows all outbound traffic
- Allows SSH (port 22) from anywhere (IPv6)
- You can modify or create new ones

## Recommended for Beta

Start with the default group. You can open additional ports (HTTP, HTTPS, custom ports) as needed for your tests.
